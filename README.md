## Blog Apps

Deploy By. Netlify

```bash
 https://ok-post.netlify.app/
```

Step by step run project

```bash
 git clone https://github.com/firmansyf/blog-app.git

 npm install or yarn install

 yarn dev
```

### Main Feature

- Create Data
- Read Data
- Update Data
- Delete Data
- Searching Data By. Name
- Pagination

### Stack

- NextJs
- Typescript
- Tailwind
- shadcn/ui
